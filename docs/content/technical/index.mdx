# Technical details of Lawtext

This section explains the technical details of Lawtext.

- [Intermediate data](./intermediate-data.mdx) - Lawtext is a plain-text format for law (mainly targeting Japanese law). The Lawtext parser converts a Lawtext string into JSON data representing the law structure based on the Standard Law XML Schema.
- [Parsing](./parsing.mdx) - The goal of the Lawtext parser is to extract the law content that the original Lawtext (plain-text) expresses and obtain it as a intermediate data structure.
- [Syntax](./syntax.mdx) - About the syntax of Lawtext.
- [Analyzing](./analyzing.mdx) - Lawtext provides some functionalities for analyzing corresponding parentheses, term definition positions, clause number references, etc.
- [Rendering](./rendering.mdx) - Lawtext provides the functionality of rendering into several formats, such as Lawtext, Standard Law XML, HTML, and docx.
